About Us

Virtual reality provides the user a completely immersive environment - allowing the user to engage with a world in the closest way possible. The only drawback to this immersion is the finality; setting up an environment is time intensive, and it is currently impossible to change the world dynamically, as it is being experienced. For our project, we will build a platform that will allow a separate individual - a moderator - to dynamically modify the world the user is experiencing on the fly from another device. This capability will allow for a truly real-time experience unlike what is possible in current virtual reality technology.
